AIS Chatbot Callflow Project

Led the development of a comprehensive Callflow management system for AIS's chatbot platform. This project involved designing conversation flows that guide customers through various service inquiries, from bill inquiries to service cancellations. I utilized Genie Dialog Design Tool to create intuitive conversation paths and orchestrated the entire project using Azure DevOps.

Architecture & Structure: The system was organized into multiple repositories for better maintainability—Web Admin portal for configuration, Static Web App for user-facing interface, and Core Engine for processing logic. All components were managed through Azure DevOps with automated CI/CD pipelines to ensure quality and rapid deployment.
