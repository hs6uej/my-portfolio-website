Project Management & Quality Standards

ในฐานะ System Analyst ผมได้รับผิดชอบในส่วนของการบริหารโครงการและควบคุมมาตรฐานคุณภาพ ซึ่งเป็นส่วนสำคัญในการพัฒนาระบบให้มีประสิทธิภาพและเป็นไปตามข้อกำหนดสากล

  • ISO 29110: ดำเนินการในฐานะผู้จัดการโครงการ (Project Manager) เพื่อวางรากฐานและพัฒนากระบวนการทางด้านซอฟต์แวร์ให้ได้มาตรฐาน
  • ISO 27001: รับหน้าที่เป็นผู้ควบคุมเอกสาร (Document Control Coordinator - DCC) ดูแลจัดการเอกสารทั้งหมดในระบบให้เป็นไปตามมาตรฐานความปลอดภัยของข้อมูล
  • System Documentation: จัดทำเอกสารสำคัญตลอดวงจรการพัฒนาโปรเจกต์ ตั้งแต่ PLANNING, Statement of Work (SOW), Software Requirement Specification (SRS), จนถึง User Acceptance Test (UAT)

Data & Backend Engineering

พัฒนาและดูแลระบบหลังบ้าน รวมถึงกระบวนการจัดการข้อมูล เพื่อสนับสนุนการทำงานของส่วนหน้าและระบบรีพอร์ต

  • ETL Process: พัฒนากระบวนการ Extract, Transform, Load (ETL) สำหรับจัดการข้อมูล โดยใช้ภาษา Python
  • Web Scraping: สร้างระบบดึงข้อมูลจากหน้าเว็บไซต์ต่างๆ แบบอัตโนมัติเพื่อนำมาจัดเก็บลงฐานข้อมูล
  • Azure Blob Management: ใช้ Azure Data Factory (ADF) ในการทำระบบ Freeze and Deploy เพื่อบริหารจัดการข้อมูลบน Azure Blob Storage
  • Reporting Backend: พัฒนา Backend ด้วย Node.js เพื่อส่งข้อมูลไปยัง Elasticsearch สำหรับนำไปสร้าง Report และ Dashboard
